/reddit

The API fetches over 100 posts from r/axolotls and sends back a number of those based on filters provided.

Params:

Name Description Possible values Default value
set What set to pick the post from.
  • new
  • hot
  • rising
  • any
 any
minScore What the lowest possible score should be. Any number ranging from -Infinity to Infinity. -Infinity
minMedia How many pictures or videos the post should have at least. Any number ranging from 0 to Infinity. 0
minAwards How many awards the post should have at least. Any number ranging from 0 to Infinity. 0
minComments How many comments the post should have at least. Any number ranging from 0 to Infinity. 0
maxScore What the highest possible score should be. Any number ranging from -Infinity to Infinity. Infinity
maxMedia How many pictures or videos the post should have at most. Any number ranging from 0 to Infinity. Infinity
maxAwards How many awards the post should have at most. Any number ranging from 0 to Infinity. Infinity
maxComments How many comments the post should have at most. Any number ranging from 0 to Infinity. Infinity
author Who the author must be. Any string, if this parameter isn't specified it won't be filtered.
flair What the author must be. Any string, if this parameter isn't specified it won't be filtered.
nsfw If NSFW posts should be allowed or not. #ref-1 0
spoiler If posts marked as spoiler should be allowed or not. #ref-1 0
archived If posts marked as archived should be allowed or not. #ref-1 1
is_crosspostable If posts marked as crosspostable should be allowed or not. #ref-1 1
pinned If pinned posts should be allowed or not. #ref-1 1
locked If locked posts should be allowed or not. #ref-1 1
Example
/reddit/getflairs

The API fetches over 100 posts from r/axolotls. These posts have flairs, each flair is logged and can be fetched with this route.

Params:

There are no parameters for this route! :D
Example
/reddit/getcount

The API fetches over 100 posts from r/axolotls. These posts can be filtered and then counted with this route.

Params:

This route uses the exact same parameters as /reddit.
Example
/reddit/fromid

Once you know the ID of a post, you can fetch information about it with this route.

Params:

Name Description Possible values Default value
id The ID of a Reddit post. Any string.
Refs

Found a #ref-x? Here you'll find their meaning.

Name Description
#ref-1 Either 0, 1, or 2. 0 means the value must be false, 1 means it doesn't matter what the value is, 2 means the value must be true.
